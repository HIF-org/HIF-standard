# Case study
[`HIF-demo.ipynb`](HIF-demo.ipynb): The tutorial used to replicate the use case experiment in [10.1017/nws.2025.10018](https://doi.org/10.1017/nws.2025.10018). It demonstrates how to use the HIF standard to facilitate data exchange between different libraries.
