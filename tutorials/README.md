# List of tutorials

These tutorials demonstrate how to use the HIF functionality available in


- Hypergraph Analysis Toolbox (HAT)  [`HAT.ipynb`](HAT.ipynb)
- Hypergraphx  [`hgx.ipynb`](hgx.ipynb)
- HyperNetX   [`hnx.ipynb`](hnx.ipynb)
- nx_hif  [`nx.ipynb`](nx.ipynb)
- TopoNetX  [`tnx.ipynb`](tnx.ipynb)
- XGI [`xgi.ipynb`](xgi.ipynb)
- SimpleHypergraphs.jl [`HIF-SimpleHypergraphs-demo.ipynb`](HIF-SimpleHypergraphs-demo.ipynb)
- The tutorial used to replicate the use case experiment in the [paper](https://doi.org/10.1017/nws.2025.10018). It demonstrates how to use the HIF standard to facilitate data exchange between different libraries [`HIF-demo.ipynb`](HIF-demo.ipynb)
